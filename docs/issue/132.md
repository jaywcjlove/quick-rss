Ollamac：一个专为 Ollama 打造的 macOS 原生应用
===

如果你在 Mac 上使用 Ollama，不妨试试 Ollamac —— 一个专为 Ollama 打造的 macOS 原生应用，支持所有 Ollama 模型，自定义 Host，内置语法高亮，界面简洁易用，完全免费且开源，非常适合开发者和 AI 用户使用。

Window Glue：一款 macOS 菜单栏工具，可将两个窗口粘连同步操作
===

一个简单的 macOS 菜单栏小工具，可以将两个窗口“粘合”在一起，使它们（大部分情况下）像一个窗口那样协同移动和调整大小。

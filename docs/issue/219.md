Nook：一款为 macOS 设计的快速、极简、侧边栏优先的浏览器。
===

一款面向 macOS 的快速极简浏览器，采用“侧边栏优先”设计。界面简洁，让你专注内容，同时利用现代 macOS 技术实现流畅且低内存的性能表现。

<img alt="Nook" src="https://github.com/user-attachments/assets/8eca7ca1-fd17-40f9-be17-6376c5bdd941" />

## 功能亮点

- 侧边栏优先的导航 —— 使用垂直标签页，切换页面更自然且不占用界面空间。
- 极速性能 —— 基于现代 macOS 技术优化，启动快、响应快且更节省内存。
- 极简现代界面 —— 去除干扰，界面以网页内容为中心。

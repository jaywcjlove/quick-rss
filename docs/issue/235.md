Keyden：一款简洁优雅的 macOS 菜单栏 TOTP 验证器
===

**Keyden** 是一款专为 macOS 打造的 **简洁、优雅且安全的菜单栏 TOTP 双因素认证器**。
它专注于高效、安全与原生体验，让你无需打开臃肿的应用，即可快速获取一次性验证码。

所有 TOTP 密钥均 **安全存储于 macOS 钥匙串（Keychain）**，数据离线优先、本地加密，真正做到隐私可控、使用安心。

### ✨ 核心特性

* 🔐 **安全存储** TOTP 密钥加密保存在 macOS Keychain，系统级安全保障
* 📋 **一键复制验证码** 点击即可复制当前验证码，快速粘贴，无多余操作
* 📷 **二维码支持** 扫描二维码快速添加账户，支持导出二维码图片备份
* 📥 **批量导入账户** 通过剪贴板或输入框一次性导入多个账号
* ☁️ **GitHub Gist 同步（可选）** 使用私有 Gist 在多台 Mac 间同步数据
* 💾 **离线优先设计** 无需联网即可使用，所有数据本地加密存储
* 🎨 **主题支持** 支持明暗模式，自动跟随系统外观
* 🌍 **多语言支持** 提供英文与简体中文界面
* 📌 **置顶与排序** 常用账户可置顶，支持拖拽自由排序
* 📂 **分组视图** 按发行商分组展示账户，管理更清晰
* ⌨️ **全局快捷键** 自定义快捷键快速唤出菜单（默认：⌘⇧K）
* 🔄 **导入 / 导出** 轻松备份和恢复所有令牌数据
* 🚀 **开机启动** 支持随 macOS 自动启动，随时可用

👉 https://github.com/tasselx/Keyden

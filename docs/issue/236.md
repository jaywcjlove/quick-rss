macUSB：面向 Apple Silicon(M1/M2/M3) 的 macOS / OS X 启动盘创建工具
===

macUSB 是一款专为 Apple Silicon（M1 / M2 / M3）Mac 打造的一键式工具，让创建 可启动的 macOS / OS X USB 安装盘 变得前所未有的简单。

它可以将你的现代 Mac 转变为一台 “系统服务机”，轻松为老款 Intel Mac 准备系统安装介质，无需折腾命令行、证书问题或复杂的磁盘操作。

## 🧩 项目背景

随着 Apple 全面转向自研芯片，
在新款 Mac 上为旧款 Intel 设备创建安装介质变得异常困难：

- ❌ createinstallmedia 报错
- ❌ 旧版安装器证书过期
- ❌ 磁盘格式与系统不兼容
- ❌ 系统工具无法识别老版本安装包

macUSB 正是为了解决这些问题而生。

## 🚀 macUSB 能做什么？

macUSB 自动完成整个流程，你只需要点击一次：

- 🔍 识别 .dmg 中的系统版本
- 💽 自动格式化并准备 USB 驱动器
- 📦 拷贝并构建可启动安装介质
- 🛠 对特定旧系统自动应用必要修复

无需查教程、无需 Terminal、无需手动修补文件。

## 🎯 你将获得什么？

- **🕰 完整的旧系统支持**：可在最新 Apple Silicon Mac 上创建 十多年前的系统安装盘，例如：OS X Lion、Mountain Lion、High Sierra 等
- **⚡ 显著节省时间**：自动识别系统、格式化磁盘并完成复制，全程无需干预
- **🔧 自动修复机制**：针对 High Sierra / Mojave 等系统，自动处理因 Apple 证书过期导致的安装失败问题

PortKiller：一款用于查找并终止端口进程的 macOS 菜单栏应用。
===

这是一款原生 macOS 菜单栏应用，用于快速查找并终止占用端口的进程。
非常适合开发者在本地开发过程中，快速释放 **3000**、**8080**、**5173** 等常用端口，无需再使用终端命令。

Osaurus：原生 Apple Silicon LLM 服务器，支持菜单栏、命令行和 macOS 26 的 Apple 模型。
===

**Osaurus** 是一款原生 macOS 的一体化 **LLM 服务器**，支持 **MCP（Model Context Protocol）**。
它可以在 Apple Silicon 上同时运行 **本地与远程大语言模型**，并提供与 **OpenAI 兼容的 API**、工具调用能力以及内置插件生态。

## 什么是 Osaurus？

Osaurus 将多个关键能力整合到一个统一的 macOS 应用中，包括：

* **MLX 运行时** —— 基于 MLX，为 Apple Silicon 优化的本地模型推理
* **远程模型提供商** —— 可连接 OpenAI、OpenRouter、Ollama、LM Studio 以及任何兼容 OpenAI API 的服务
* **OpenAI / Anthropic / Ollama API 兼容** —— 可直接作为现有工具的替代后端，无需改代码
* **MCP 服务器** —— 通过 Model Context Protocol 向 AI Agent 暴露工具能力
* **远程 MCP 提供商** —— 聚合外部 MCP 服务器的工具，统一管理
* **插件系统** —— 支持社区插件和自定义工具扩展
* **开发者工具** —— 内置请求分析、服务探索器，便于调试
* **Apple Foundation Models** —— 在 macOS 26+（Tahoe）上直接使用系统级模型

## 功能一览

| 功能           | 描述                                        |
| ------------ | ----------------------------------------- |
| 本地 LLM 服务器   | 本地运行 Llama、Qwen、Gemma、Mistral 等模型         |
| 远程模型支持       | OpenAI、OpenRouter、Ollama、LM Studio 或自定义接口 |
| OpenAI 兼容接口  | `/v1/chat/completions`，支持流式输出与工具调用        |
| Anthropic 兼容 | `/messages` 接口，可直接用于 Claude Code 与官方 SDK  |
| MCP 服务器      | 连接 Cursor、Claude Desktop 等 MCP 客户端        |
| 远程 MCP       | 聚合多个 MCP 服务器的工具能力                         |
| 工具与插件        | 浏览器自动化、文件系统、Git、Web 搜索等                   |
| 自定义主题        | 创建、导入、导出主题，支持完整配色定制                       |
| 开发者工具        | 请求洞察、API Explorer、实时接口测试                  |
| 菜单栏聊天        | 快捷聊天浮窗，支持会话历史与上下文追踪（⌘;）                   |
| 模型管理         | 从 Hugging Face 下载并管理本地模型                  |

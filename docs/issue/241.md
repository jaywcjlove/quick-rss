Alertivity：菜单栏中的实时系统健康状态
===

**Alertivity** 是一款 macOS 菜单栏工具，它会定期采样 CPU、内存、磁盘、网络和进程数据，帮助你及早发现系统问题。它将系统健康状态总结为 **正常**、**升高** 或 **严重**，突出显示触发该状态的指标，并提供对问题进程的快速操作。

### 特性

* **实时系统采样**：每几秒钟更新一次系统状态，状态由 CPU、内存、磁盘和网络的最高严重性指标驱动（带有轻微滞后避免频繁切换）。
* **菜单栏指示器**：显示总体状态或特定指标（CPU、内存、磁盘、网络）；可选的自动切换仅在严重状态时生效，切换优先级为 **CPU > 内存 > 磁盘 > 网络**，并在状态变化时稍作延迟，同时在状态非严重时可隐藏指示器。
* **紧凑菜单**：显示每个指标的总结、网络吞吐量以及 CPU/内存/网络的高活动进程，提供快速操作，可直接在活动监视器中查看或终止进程（磁盘采样被排除以避免延迟）。
* **通知**：对于严重状态或高活动进程，提供通知，包括触发指标/数值和进程元数据；通知会进行限速以避免过多提醒。
* **设置选项**：可设置 Dock 可见性、开机启动、菜单图标行为（默认图标类型、自动切换、仅在高活动时显示、显示指标图标）、通知选项以及检测阈值（CPU 百分比及持续时间要求）。

通过 **Alertivity**，你可以更轻松地监控和管理 macOS 系统的健康状态，及时发现并处理潜在问题，提升系统稳定性。

Seanime：一款跨平台的媒体服务器，支持流式播放动漫和漫画管理
===

Seanime 是一款功能强大的媒体服务器，提供网页界面和桌面应用，帮助用户管理本地库、流式播放动漫，并提供漫画阅读功能。

## 重要说明

* Seanime 不提供、托管或分发任何媒体内容，所有内容均需用户自行合法获取。
* 用户需要遵守当地的法律法规，应用内列出的扩展与 Seanime 无关，如违反版权法，扩展可能会被移除。

## 主要功能

* **跨平台支持**：网页界面和桌面应用，方便在不同设备上使用。
* **内置视频播放器**：支持 Seanime Denshi（包括 SSA/ASS 字幕，Anime4K 锐化等）。
* **完美集成 AniList**：可以浏览、管理、评分、发现和更多。
* **自定义源支持**：允许添加不在 AniList 上的系列，甚至非动漫和漫画系列。
* **离线模式**：无需互联网连接也能使用。
* **扫描本地动漫库**：几秒钟内扫描所有动漫，不需要特殊的文件夹结构或重命名。
* **扩展的 Torrent 搜索引擎**：方便查找和下载资源。
* **支持直接流式播放**：使用 Bittorrent、Torbox 和 Real-Debrid，直接将 torrent 内容流式传输到播放器。
* **自动下载新剧集**：通过自定义过滤器自动下载。
* **桌面端观看支持**：支持 MPV、VLC 和 MPC-HC 等播放器。
* **移动端观看支持**：通过外部播放器链接，在手机上观看。
* **支持转码和直接播放**：可以流式传输到任何设备的浏览器。
* **漫画阅读**：支持从本地库或扩展中阅读漫画章节。
* **跟踪剧集**：可以查看即将播出的剧集，或是已经错过的剧集。
* **可定制的 UI**：根据个人需求调整界面。

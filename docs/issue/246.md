BetterAudio：让 macOS 的音频管理更简单
===

**BetterAudio** 是一款 macOS 菜单栏音频管理应用，专为需要更高效、更精细音频控制的用户打造。
无需打开系统设置，你就可以直接在菜单栏中完成所有常用音频操作。

<img alt="BetterAudio" src="https://github.com/user-attachments/assets/838608db-cd50-463d-9164-e804a0aee9c6" />

BetterAudio 让你能够快速切换输入与输出设备、精确控制音量、查看当前正在播放的内容，并为常用操作设置全局快捷键，一切都在菜单栏中完成，简单直观。

### 核心功能

* 点击菜单栏图标即可快速静音
* 多种麦克风静音状态图标，清晰显示当前状态
* **正在播放（Now Playing）**：显示当前播放的曲目
* 曲目切换时的自定义通知
* 强制媒体按键作用于指定播放器（如 Spotify、Apple Music）
* 将音量键重定向到指定播放器
* 快速切换音频输入与输出设备
* 显示所有可用音频设备，并支持隐藏或加入黑名单
* 直接从菜单栏调节系统音量
* 调整音量或切换设备时的自定义通知
* 自定义菜单栏中显示的组件内容

BetterAudio 非常适合播客创作者、音乐爱好者、远程会议用户以及对 macOS 音频控制有更高需求的用户，让音频管理变得高效、直观且可控。

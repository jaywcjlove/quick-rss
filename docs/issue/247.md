Quotio：统一管理 AI 订阅并智能切换工具
===

Quotio 是一款原生 macOS 应用，用于管理 **CLIProxyAPI**——一个为 AI 编程代理提供支持的本地代理服务器。它帮助你在一个地方管理多个 AI 账号、跟踪配额，并配置 CLI 工具。

<img alt="Quotio" src="https://github.com/user-attachments/assets/93e479f9-c755-4f05-8bb6-4479e6f2ebc1" />

## 功能特点

- 🔌 **多提供商支持**：通过 OAuth 或 API Key 连接 Gemini、Claude、OpenAI Codex、Qwen、Vertex AI、iFlow、Antigravity、Kiro 和 GitHub Copilot 账号。
- 🚀 **一键代理配置**：自动检测并配置 Claude Code、OpenCode、Gemini CLI 等 AI 编程工具。
- 📊 **实时仪表盘**：实时监控请求流量、Token 使用情况及成功率。
- 📉 **智能配额管理**：按账号可视化配额追踪，并提供自动切换策略（轮询 / 先填满）。
- 🔑 **API Key 管理**：生成并管理本地代理的 API Key。
- 🖥️ **菜单栏集成**：从菜单栏快速访问服务器状态、配额概览和自定义提供商图标。
- 🔔 **通知提醒**：配额不足、账号冷却期或服务异常时发送提醒。
- 🔄 **自动更新**：内置 Sparkle 更新器，实现无缝更新。
- 🌍 **双语支持**：提供英文和越南语界面。

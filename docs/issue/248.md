TuxMate：Linux 系统应用一键批量安装助手
===

TuxMate 是一款网页工具，可为不同 Linux 发行版生成批量安装脚本，让新系统的应用配置轻松又快速。

<img alt="TuxMate" src="https://github.com/user-attachments/assets/13de7a10-6685-461d-bd7a-5851d73a1889" />

<img  alt="TuxMate" src="https://github.com/user-attachments/assets/ee9a8fd7-1973-4938-a6be-da5482022411" />

📦 **支持发行版**：Ubuntu / Debian、Arch Linux、Fedora、openSUSE、NixOS 以及 Flatpak 和 Snap。

## **主要功能**：

* 150+ 应用覆盖浏览器、开发工具、办公、游戏等
* 自动检测已安装软件、智能生成安装脚本
* 并行安装 Flatpak、AUR 自动处理
* 彩色输出与进度条、安装总结报告
* 显示可用应用及不可用应用说明

轻松配置新系统，只需一键，省时又高效！

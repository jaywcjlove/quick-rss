ipatoolUI：优雅的 IPA 下载工具，轻松获取 iOS 应用安装包
===

ipatoolUI 是一款基于 ipatool 的图形界面应用，让您能够直接从 App Store 下载 iOS 应用的 IPA 安装包。告别命令行的复杂操作，通过直观友好的界面，只需几次点击即可获取任何 App Store 应用的安装包文件。

<img alt="ipatoolUI" src="https://github.com/user-attachments/assets/591f2690-19c1-4fb1-a4f8-e8e86a173809" />

## 主要特性

- **图形化界面** - 为 ipatool 命令行工具提供现代化的 GUI，操作简单直观
- **直接下载 IPA** - 从 App Store 直接下载应用的安装包文件到本地
- **搜索与浏览** - 在 App Store 中搜索应用，查看详细信息后一键下载
- **Apple ID 登录** - 安全地使用您的 Apple ID 凭证进行认证
- **批量管理** - 支持管理和下载多个应用的 IPA 文件
- **跨平台支持** - 适用于 macOS、Windows 和 Linux

## 使用场景

- **应用备份** - 保存已购买应用的本地副本，防止应用下架
- **版本控制** - 保留特定版本的应用，避免不喜欢的更新
- **侧载安装** - 用于 AltStore、Sideloadly 等工具的应用侧载
- **开发测试** - 开发者研究和分析应用结构
- **跨设备安装** - 在多台设备上轻松部署相同版本的应用

## 工作原理

[ipatool](https://github.com/majd/ipatool)UI 基于强大的 [ipatool](https://github.com/majd/ipatool) 命令行工具构建，通过图形界面封装其核心功能。它使用您的 Apple ID 凭证与 App Store 进行合法交互，下载您有权访问的应用安装包。

## 快速开始

1. 下载并安装 ipatoolUI
2. 使用您的 Apple ID 登录
3. 搜索或浏览想要下载的应用
4. 点击下载按钮获取 IPA 文件
5. 使用您喜欢的工具安装或管理 IPA 文件

## 注意事项

- 需要有效的 Apple ID 账户
- 只能下载您账户有权访问的免费应用或已购买的付费应用
- 请遵守 Apple 的服务条款和当地法律法规
- 仅供个人合法使用

RepoBar：在 macOS 菜单栏一览 CI、PR 和发布状态
===

RepoBar 是一款 macOS 菜单栏应用，让你可以快速了解 GitHub 仓库的健康状态。使用 Swift 6.2 和 Xcode 26 构建，它可以显示关键信息，包括：

* CI 状态
* 未处理的 issue 和 pull request
* 最新发布版本
* 最近评论
* 访问量（独立访客）
* 自定义提交热力图

<img alt="RepoBar" src="https://github.com/user-attachments/assets/6095c589-a592-4132-afd2-74adb0b61f50" />


**功能亮点：**

* **MenuBarExtraAccess**：区分左键和右键点击，执行不同操作。
* **Sparkle**：自动更新应用，体验无缝升级。
* **基于 PKCE 的浏览器登录**：支持 GitHub.com 和 GitHub Enterprise，安全便捷。

RepoBar 让你无需打开浏览器，也能随时掌握仓库状态，提高开发效率。

👉 https://github.com/steipete/RepoBar

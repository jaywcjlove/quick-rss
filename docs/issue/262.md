WailBrew：为 macOS 提供现代化 Homebrew 图形界面管理器
===

WailBrew 是一款 **为 macOS 设计的极简且直观的 Homebrew 图形界面管理工具**，让你无需终端命令即可轻松管理 Homebrew 软件包。

<img alt="WailBrew" src="https://github.com/user-attachments/assets/1d5570ce-57bb-4d34-ae1b-f1bcefdacb59" />

通过 WailBrew，你可以：

- 查看和管理已安装的 **Homebrew 公式（Formulas）和应用包（Casks）**
- 使用即时过滤搜索软件包列表
- 一键安装、卸载和升级软件包
- 检查过时的软件包，并支持单个或全部更新
- 查看详细的软件包信息，包括依赖关系和冲突
- 管理 Homebrew 软件源（tap / untap）

WailBrew 的设计灵感来自 **Cakebrew**，但采用了更现代的 UI 设计，结合 **Go、Wails 和 React** 构建，提供兼顾性能与体验的原生应用。

你可以通过 Homebrew 快速安装并开始使用：

```bash
brew tap wickenico/wailbrew
brew install --cask wailbrew
````

WailBrew 支持多种 macOS 版本，兼容 Apple Silicon 与 Intel 平台，并提供多语言界面（包括简体中文和繁体中文）。

👉 https://github.com/wickenico/WailBrew

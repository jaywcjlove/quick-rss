Fresh：Vim 和 Nano 的替代品，全新外观的代码编辑器！
===

Fresh 是一个全新的、轻量级的开源终端代码编辑器，它为开发者提供了极致的性能和简单的使用体验。用 Rust 开发，Fresh 提供了超快的启动和响应时间，同时支持鼠标操作、快捷键和命令面板，完美适配 Windows、macOS 和 Linux 系统。

## 功能：

* 100% 在终端中运行
* 基于 Rust 开发，极快的响应速度
* 支持鼠标、快捷键和命令面板
* 适用于 Windows、macOS 和 Linux

👉 https://github.com/sinelaw/fresh

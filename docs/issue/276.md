RzWeb：基于浏览器的逆向工程平台，它完全通过 WebAssembly 在浏览器中运行
===

RzWeb 将 Rizin 的全部功能带入浏览器。您将获得一个完整的终端，可以运行任何 Rizin 命令，并且还提供专门的视图来查看反汇编、控制流图、十六进制转储和字符串。所有操作都在本地处理，您的文件永远不会离开您的设备。

### 主要功能：

* **终端**：集成的终端让您可以直接访问 Rizin 的命令行界面。您可以运行 `pdf` 反汇编函数、`afl` 列出所有函数、`px` 查看十六进制转储，或任何您在正常 Rizin 会话中会使用的命令。命令之间可以用分号连接，例如 `s main;pdf`。

* **反汇编**：具有语法高亮的汇编代码，并提供地址跳转。点击地址可跳转，查看交叉引用，并追踪您当前在二进制文件中的位置。

* **控制流图**：可视化显示函数结构，查看基本块的连接，识别循环，并一眼了解控制流。

* **十六进制视图**：查看原始字节，导航到任何偏移并直接检查二进制数据。

* **字符串**：从二进制文件中自动提取的字符串，适用于查找硬编码路径、错误信息、加密密钥和其他有趣的数据。

### 支持的格式：

RzWeb 支持所有 Rizin 支持的格式：

* **ELF** - Linux 可执行文件和共享库
* **PE/PE+** - Windows 可执行文件和 DLL
* **Mach-O** - macOS 和 iOS 二进制文件
* **Raw** - 固件、转储，其他任何格式

通过 RzWeb，您可以在浏览器中轻松实现强大的逆向工程分析，而无需任何安装或配置。

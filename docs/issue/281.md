Reminders MenuBar：简单的 macOS 菜单栏应用，用于查看和互动提醒事项，管理 Apple Reminders
===

`Reminders Menubar` 是一个轻量级的 macOS 菜单栏应用，它使您能够轻松管理和查看您的 Reminders（提醒事项）任务。它在菜单栏提供简洁的界面，帮助您随时随地查看、添加和管理提醒事项，无需打开完整的 Reminders 应用。

<img  alt="Reminders MenuBar" src="https://github.com/user-attachments/assets/c787a00b-1a10-459d-baeb-53b7f6c4e611" />

## 特性

* **任务管理**：查看、添加和标记任务。
* **简洁界面**：在菜单栏中轻松访问 Reminders。
* **多帐户支持**：如果您有多个 iCloud 账户，它支持切换和管理每个账户的提醒事项。
* **任务过滤**：快速筛选任务，根据优先级、日期等条件进行过滤。
* **同步**：所有数据实时与 iCloud 同步，确保信息保持更新。

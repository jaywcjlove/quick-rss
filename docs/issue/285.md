CodexMonitor：专为开发者设计的本地 Codex AI 智能体编排与协作中心
===

**CodexMonitor** 是一款基于 Tauri 开发的 macOS 应用，其核心定位是作为多个 **Codex 智能体** 的“指挥部”。它不仅是一个聊天界面，更是一个深度集成工作区、Git 和终端的复杂开发环境。

<img alt="CodexMonitor" src="https://github.com/user-attachments/assets/46f98f77-9634-4d87-b61f-e6c62bd64550" />

### 核心特性

* **多工作区管理：** 支持添加并持久化本地工作空间，每个工作区都能启动独立的 `codex app-server`。应用可以自动恢复对话线程，并追踪 Agent 的运行状态。
* **深度 Git & GitHub 集成：** * 内置差异对比（Diff）、暂存（Stage）及提交记录查看。
  * 支持通过 `gh` 工具直接在应用内管理 GitHub Issues 和 PR。
  * **“Ask PR” 功能**：能将整个 PR 的上下文直接发送给 AI 进行分析。
* **高级 Composer (输入框)：**
  * 支持图片附件、拖放和粘贴。
  * 具备强大的**自动补全**功能：支持技能（`$`）、提示词模板（`/prompts`）、代码审查（`/review`）和文件路径（`@`）。
* **智能体隔离与克隆：** 支持 Worktree 功能，为不同的 AI 任务创建隔离的工作目录，确保实验性的修改不会污染主分支。
* **开发者工具箱：**
  * **文件树系统：** 支持文件搜索、类型图标显示及在 Finder 中定位。
  * **集成终端：** 实验性的多标签终端窗口，可直接运行后台命令。
  * **语音输入：** 集成 Whisper 协议，支持长按录音（Hold-to-talk）并显示动态波形。
* **极致的 macOS 交互体验：** 采用原生效果的毛玻璃（Vibrancy）侧边栏，支持高度自定义的面板布局（调试面板、终端面板、计划面板等），并提供账户额度实时监控。

### 适用场景

它是为那些希望深度将 AI（如 OpenAI Codex 或类似模型）集成进日常开发工作流的用户准备的。如果你厌倦了在浏览器和编辑器之间不停切换，CodexMonitor 提供了一个将“代码理解、任务规划、Git 操作、AI 对话”融为一体的单窗口解决方案。

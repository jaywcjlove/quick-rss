Itsyhome：菜单栏应用，用于控制HomeKit智能家居设备
===

ItsyHome 是一款为 macOS 原生打造的菜单栏应用，旨在让你无需离开当前工作流即可掌控全屋智能设备。

<img alt="Itsyhome" src="https://github.com/user-attachments/assets/39e67c0c-64ca-493a-aeb2-a689e05fd9f4" />

- 全能控制：深度支持灯光、插座、窗帘、温控器及安防系统等几乎所有 HomeKit 设备类目。
- 极致原生：基于 AppKit 构建，提供纯正的 macOS 视觉体验；支持将特定设备、场景或房间直接**固定（Pin）**到菜单栏，并自定义快捷键。
- 高效联动：不仅支持执行场景，还能利用 DeepLinks、Webhooks 和 CLI 工具，将智能家居与 Shortcuts、Raycast 或 Stream Deck 完美融合 (Pro)。
- 实时监控：Pro 版本支持摄像头实时流预览，并在门铃响起时自动弹出画面，配合覆盖按钮即可一键开锁或开灯。
- 多端同步：通过 iCloud 在多台 Mac 间同步你的偏好设置与自定义分组。

👉 https://github.com/nickustinov/itsyhome-macos

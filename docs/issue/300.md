ClearDisk：一键找回被开发缓存占用的 50–500GB Mac 磁盘空间
===

你的 Mac 可能正被大量 **开发者缓存**悄悄占满。仅 **Xcode DerivedData** 就可能占用 **200GB+**，再加上 npm、Homebrew、Docker、pip、Cargo 等工具的缓存，很容易丢掉 **几十到几百 GB** 的 SSD 空间。

**ClearDisk** 是一款 **免费、开源的 macOS 菜单栏工具**，专门用于监控和清理开发者缓存。
仅 **590KB、零依赖、无网络访问、无数据收集**，完全本地运行。

✨ 主要功能：

* 🔍 扫描 **44 种开发缓存**（Xcode、npm、Homebrew、Docker、pip、Cargo、Go、Gradle 等）
* 📊 菜单栏实时监控磁盘占用，空间紧张时提醒
* 🧠 智能建议：识别长期未使用缓存并推荐清理
* 🛡 风险等级提示：安全 / 谨慎 / 高风险
* ♻️ 安全删除：文件先进入废纸篓，可随时恢复
* 📦 项目构建产物扫描（node_modules、target、build 等）

如果你的 Mac **经常磁盘爆满**，或者你是 **iOS / Web / 后端开发者**，ClearDisk 可以帮你快速找回被遗忘的磁盘空间。

开源项目，完全透明。
让开发缓存不再偷偷吃掉你的 SSD。

👉 https://github.com/bysiber/cleardisk

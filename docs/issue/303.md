Remodex：通过本地优先桥接在 iPhone 上远程控制 Codex 的工具，支持配对会话与远程操作
===

Remodex 是一个 **本地优先（local-first）的开源桥接工具 + iOS 应用**，让你可以直接通过 iPhone 控制运行在 Mac 上的 Codex。

你的 Mac 会继续在本地运行 Codex，而 iPhone 通过 **配对的 WebSocket 中继会话**连接，从而让你随时随地管理任务和工作流。

### 核心功能

• **端到端加密**的 iPhone 与 Mac 配对和聊天
• **Fast Mode**：降低延迟，加快响应速度
• **Plan Mode**：在执行前进行结构化任务规划
• **运行中控制**：无需重启即可调整当前任务
• **队列提示**：在任务运行时提前加入下一条指令
• **通知提醒**：任务完成或需要操作时即时通知
• **手机端 Git 操作**：commit / push / pull / 切换分支
• **推理强度控制**：调节 Codex 的思考深度
• **访问权限控制**：支持 On-Request 或 Full Access
• **照片附件**：直接从相机或相册添加
• **二维码配对**：自动重连
• **线程历史同步**：与 Mac 上的 Codex 共享会话记录

### 中继服务

目前完整的 **手机到 Mac 的连接流程**仍依赖托管中继服务 **api.phodex.app**。

在当前 **TestFlight 测试阶段**，该服务免费提供，用于验证功能和优化体验。

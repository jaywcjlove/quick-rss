TouchBridge：不用花 $199 买键盘，用手机给 Mac 加 Touch ID
===

TouchBridge 可以让你的手机指纹，直接用来解锁 Mac。sudo、锁屏、App Store —— 不用再一遍遍输入密码。

![TouchBridge](https://github.com/user-attachments/assets/56142da4-8cd3-4d7e-a861-68ef41de5ee4)

问题是：Apple 把 Touch ID 当成“付费功能”。

Mac Mini、Mac Studio、Mac Pro —— 完全没有指纹。  
部分 MacBook 也需要更高配置才有。  
iMac 甚至要额外买带指纹的键盘。  

结果就是：你每天都在重复输入密码。

TouchBridge 的解决方案：

✔ 用手机指纹解锁（支持 iPhone / Android）  
✔ 支持 sudo、锁屏等场景  
✔ 不需要 $199 的 Magic Keyboard  

你的手机已经有指纹识别，现在 Mac 也可以用了。

👉 https://github.com/HMAKT99/UnTouchID

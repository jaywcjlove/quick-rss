SuperCmd：把 Raycast + 语音 + AI 全部融合在一个开源启动器里
===

发现一个很有意思的 macOS 开源启动器 —— SuperCmd 👇

<img alt="SuperCmd" src="https://github.com/user-attachments/assets/1bd3f650-06a2-4146-97be-acdcc5086554" />

把 Raycast + Wispr Flow + Speechify + AI + Memory 全部整合在一起：

- 兼容 Raycast 扩展生态（支持 @raycast/api）
- 支持语音输入（按住说话 → 自动转文字）
- 文本朗读（类似 Speechify）
- 内置 AI（支持 OpenAI / Ollama / Anthropic 等）
- 支持记忆能力（Supermemory）
- 脚本命令 + 扩展安装/更新
- 原生 macOS 能力（快捷键 / 颜色选择 / 片段等）

而且还带：
语法高亮、文件树、标签页、行号、状态栏 —— 全部不到 1MB ⚡️

Electron + React 构建，但主打开源 + 可扩展生态。

一个方向很野的 Raycast 替代方案 👍

👉 https://github.com/SuperCmdLabs/SuperCmd

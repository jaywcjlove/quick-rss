OpenScreen：免费开源的 Screen Studio 替代方案（录屏 + 演示工具）
===

OpenScreen 是一个面向 macOS 的开源录屏与演示工具，定位为 Screen Studio 的轻量替代方案。它不追求复杂功能，而是专注于产品演示和录屏的核心体验，让你用更低成本（甚至免费）完成高质量的视频录制与编辑。

<img alt="OpenScreen" src="https://github.com/user-attachments/assets/889368a7-f420-4e53-8f9c-4b4dd1ec41ca" />

- 支持录制窗口或整个屏幕
- 自动 / 手动缩放（Zoom）效果
- 麦克风 + 系统音频录制
- 视频裁剪与片段剪辑
- 支持标注（文字 / 箭头 / 图片）
- 支持不同片段速度调整
- 背景自定义（壁纸 / 纯色 / 渐变）
- 支持多分辨率与比例导出

如果你只需要一个**简单、好用、不收费**的录屏工具，这个很值得试试 👍

👉 https://github.com/siddharthvaddem/openscreen

GhostVM：Apple Silicon 上最优雅的 macOS 虚拟机工具
===

基于 Virtualization.framework，原生性能极致；每个 VM 是一个可随意复制的 .GhostVM 独立 bundle。支持快照、克隆、深度主机集成、剪贴板共享、端口转发，并提供完整 CLI 与 API，完美适合 AI 沙箱、开发测试和隔离工作流。

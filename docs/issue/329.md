Gridex：原生 macOS/Windows 数据库内置 AI 对话的原生数据库 IDE，支持 PostgreSQL、MySQL、SQLite、Redis、MongoDB 和 SQL Server
===

用原生 Swift 做一个 macOS 数据库 IDE，支持 6 种数据库，内置 AI 写 SQL 助手。

<img alt="Gridex" src="https://github.com/user-attachments/assets/a08ce470-bab9-489d-8c0e-54b2a8b688c9" />

Gridex 是一款原生 macOS/Windows 数据库 IDE，Swift + AppKit 构建。支持 PostgreSQL、MySQL、SQLite、Redis、MongoDB、SQL Server 六种数据库直连，配备内置 AI 助手（Claude、GPT、Gemini、Ollama），直连 API 不走代理，Keychain 存储凭证。

👉 https://github.com/gridex/gridex

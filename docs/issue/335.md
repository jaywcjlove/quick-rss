MarkEdit：免费开源的 macOS 原生 Markdown 编辑器
===

免费开源的 macOS 原生 Markdown 编辑器。仅4MB超轻量体积，可流畅处理百万行文档。它类似系统自带的文本编辑（TextEdit），却专为 Markdown 优化，在体积、速度、系统原生适配与编辑准确性之间实现完美平衡。

简洁、快速、稳定。

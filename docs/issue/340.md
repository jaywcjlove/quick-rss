Radix：快速磁盘空间分析工具，可扫描文件夹或磁盘卷，用交互式旭日图查看占用并清理存储空间
===

Radix 是基于 Swift/SwiftUI 构建的原生 macOS 开源应用，主打高速磁盘扫描与可视化分析。软件采用迭代式遍历算法，搭配系统原生接口，能够高效处理深层目录结构，轻松应对海量文件扫描任务，并实时反馈扫描状态。

<img alt="Radix" src="https://github.com/user-attachments/assets/5e0c83f9-79a5-4c90-9c37-fadba4d1c2c5" />

软件创新性地使用**交互式旭日图**作为核心视图，将层级目录与文件占用情况图形化展示，支持点击下钻、悬停查看元数据。功能层面支持文件多维度排序、类型过滤，联动系统访达实现文件定位、删除、路径复制等操作，适配开发者与普通用户的文件管理需求。

<img alt="Radix" src="https://github.com/user-attachments/assets/0e16d2cb-4100-48f7-8b3a-af78b8a94a76" />

在隐私设计上，工具全程离线运行，无后台遥测、无账号体系，数据仅留存于本地设备。开源且永久免费的特性，搭配原生应用的流畅体验，让 Radix 成为 macOS 平台磁盘分析领域亮眼的开源项目。

👉 https://github.com/colinvkim/Radix

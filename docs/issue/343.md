FaceGate：使用人脸识别、Touch ID 或密码锁定敏感应用，所有处理均在设备端完成
===

FaceGate 是一款原生 macOS 开源应用锁工具，可为 Mac 带来面容解锁功能，支持通过面容、触控ID或安全密码加密各类敏感软件，所有识别运算均本地处理、无任何数据上传；系统原生不支持单独限制应用，市面同类工具均缺少人脸识别且功能单薄，而该软件完全免费开源、无恶意程序、所有操作本地执行，控制权完全交由用户。

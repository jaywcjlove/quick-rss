Hangar：轻量菜单栏工具，将终端工作流转化为一键式可切换代码片段
===

一款轻量原生 macOS 菜单栏工具，将日常终端工作流整合为一键切换的任务脚本。纯 Swift + AppKit 开发，无 Electron 冗余框架，体积轻巧无多余占用。

## 解决的痛点

开发者每日需要反复执行各类固定后台任务：不同项目的编译、文件监听脚本、交互式终端工具。每项任务对应独立工作目录与指令，分散在多个虚拟桌面终端窗口中。
会话分散难以管理，每次重启电脑后，所有任务都需要手动重新启动，操作繁琐低效。

Hangar 将所有自定义工作流统一收纳在常驻菜单栏面板内。一键开关即可唤起 iTerm2 / 系统终端并自动执行预设指令；再次关闭可平稳终止进程。支持标记重要任务，开机自动恢复运行。

## 核心功能

### 仅菜单栏常驻设计
始终置顶菜单栏快速调取，不生成坞站图标，不占用桌面视觉空间。
### 终端指令快捷脚本
一键唤起终端，自动运行提前配置好的 Shell 命令。
### 动态变量占位符
指令内可嵌入 `{{名称}}` 变量，执行前弹出输入框实时预览完整命令，自动缓存历史输入内容方便复用。
### 应用快速启动脚本
创建软件直达快捷方式，可附带参数直接打开指定项目文件夹、工程文件（例如编辑器一键加载项目目录）。
### 脚本分组管理
可自定义分类分组，批量收纳项目、工具、服务类脚本，界面整洁便于查找。

## 适用人群
独立开发者、后端/前端工程师、运维人员、经常使用终端操作 Mac 的办公人群。

## 产品优势
1. 原生系统开发，内存占用极低，启动无延迟；
2. 完全贴合 macOS 交互逻辑，无第三方臃肿依赖；
3. 轻量化管理各类命令与软件启动流程，大幅减少重复操作；
4. 开机自启任务自动恢复，重启无需重复配置。

TREK：自己搭一个旅行规划器，一站式旅行规划工具
===

自己搭一个旅行规划器，一站式旅行规划工具，支持实时协作、互动地图、预算管理、打包清单等，替代商业旅行 App。

基于 NestJS + React 构建，支持实时协作（WebSocket 同步）、互动地图（Leaflet/Mapbox GL）、行程拖拽规划、费用分摊、打包清单、文档附件、PDF 导出等功能。

👉 https://github.com/mauriceboe/TREK

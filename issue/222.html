AIassistant：你的智能 macOS AI 助手

AIassistant 是一款功能强大的原生 macOS 应用，让 AI 助手无缝融入你的工作流程。 它结合了 Google Gemini 与 OpenAI GPT 的强大能力，在桌面上为你提供智能、上下文感知的辅助体验，随时随地高效工作。

主要功能

  • 即时唤醒： 双击 右侧 Shift 键，即可在任意界面快速召唤 AIassistant。
  • 完整聊天界面： 支持与 AI 进行丰富对话，涵盖 文本、图片、视频、PDF 和 URL 等多种内容形式。
  • 原地重写： 在任意应用中直接改写或优化文本，无需复制粘贴。
  • 快捷操作： 一键执行 总结、翻译、简化 等常用 AI 操作。
  • 图像生成： 使用 Gemini 的 AI 能力 创建或编辑图像。
  • 截图分析： 一键截图，并让 AI 分析任意应用窗口的内容。
  • 多模型支持： 可在 Gemini 与 OpenAI 模型之间动态切换。
  • 玻璃质感界面： 优雅、现代的 macOS 原生设计。
  • 安全存储： 所有 API Key 安全保存于 macOS 钥匙串。
  • 无障碍优先： 完整支持键盘操作与系统级自动化，提升使用体验。

zsweep：一款为开发者设计的Vim风格扫雷网站

一款极简主义、支持全键盘操作的扫雷游戏。它最大的特色是集成了 Vim 操作语法（如 hjkl 移动、w/b 单词级跳转、/ 搜索数字等），让开发者在休闲时刻也能保持“手感”和 Flow 状态。

亮点

  • Vim 语法驱动：不仅支持基础的 hjkl 方向移动，还深度还原了 Vim 的逻辑——支持数字前缀（如 5j 向下移动 5 格）、w/b 跳转至下一个/上一个未开块、以及通过 / 搜索并跳转至特定数字。
  • 极简审美：受 Monkeytype 启发，提供多种极简配色方案，无广告、无干扰，专注于纯粹的扫雷体验。
  • 高性能架构：基于 Svelte 5 (Runes) 和 SvelteKit 构建，确保了极低的输入延迟和丝滑的响应速度。
  • 实时竞技：集成 Supabase 后端，支持全球实时排行榜，并解决了大规模数据统计下的分页统计难题。
  • 开源贡献友好：项目结构清晰，目前已有超过 1,000 名活跃用户，且在 GitHub 上整理了多个适合新手参与的 Issue。

👉 https://github.com/oug-t/zsweep

PastePaw：一款为 Windows 打造的精美剪贴板历史管理工具

一款为 Windows 打造的精美剪贴板历史管理工具，基于 Rust + Tauri + React + TypeScript 构建。

功能特性

  1. 🔒 隐私安全 - 这一点至关重要！所有数据仅存储在本地，绝不上传。
  2. 🎨 精美界面 - 现代化的深色/浅色主题，支持即时切换，配合流畅的动画效果。
  3. ⚡ 轻量极速 - 基于 Rust 构建，性能强劲，资源占用极低。
  4. ⌨️ 自定义快捷键 - 设置您习惯的快捷键来随时呼出历史记录。
  5. 📋 历史记录 - 自动保存您复制的所有内容。
  6. 🖥️ 多显示器支持 - 智能跟随，始终在鼠标所在的活动显示器上显示。
  7. 🔍 快速搜索 - 即使历史记录再多，也能秒级找到复制过的内容。
  8. 📁 文件夹管理 - 将常用的剪贴内容分类整理到自定义文件夹。
  9. 🚫 应用屏蔽 (排除列表) - 自动忽略来自特定敏感应用（如密码管理器）的内容，保护隐私。
  10. 🔄 无限滚动 - 无缝浏览无限的历史记录，无需翻页。
  11. 🛡️ 智能过滤 - 采用智能防抖逻辑，自动忽略来自其他剪贴板工具（如翻译软件、输入法）的“幽灵复制”。
  12. 🤖 AI 智能助手 - 内置 AI 功能，支持一键总结、翻译、解释代码以及修复语法错误。
  13. ⚙️ 高度可自定义 AI - 支持在设置中完全自定义 AI 功能的名称以及系统提示词 (Prompt)，打造专属您的 AI 工作流。

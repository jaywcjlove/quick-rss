TablePro：极速原生 macOS 数据库客户端（内置 AI 助手），支持 MySQL / Postgres / SQLite / MongoDB / Redis / SQL Server / Redshift

TablePro 是一款专为 macOS 打造的原生数据库客户端，支持 MySQL、MariaDB、PostgreSQL、SQLite、MongoDB、Redis、SQL Server、Redshift 和 Oracle 等多种数据库。应用基于 SwiftUI 构建，启动速度极快，占用内存仅约 50MB，无需 Java、无需订阅、无需账号，开箱即用。

它内置功能强大的 SQL 编辑器，支持智能自动补全、Tree-sitter 语法高亮、多标签查询以及百万行 SQL 文件的流畅编辑。你还可以通过自然语言直接生成 SQL，或使用 AI 助手解释、优化和排查查询错误。AI 支持 Claude、OpenAI、Gemini、Ollama 以及任何兼容 OpenAI 的接口。

TablePro 还提供强大的数据表编辑体验：点击单元格即可编辑文本、数字、日期、JSON 等数据类型，并支持变更跟踪、撤销/重做、差异对比提交、多列排序和高级筛选。

通过 SSH 隧道和 SSL/TLS 加密连接，你可以安全访问远程数据库。同时支持数据导入导出（CSV、JSON、SQL、XLSX）以及自动列映射，方便数据迁移和分析。

$ brew install --cask tablepro

简洁、高效、原生体验——TablePro 让数据库管理变得更轻量、更智能。

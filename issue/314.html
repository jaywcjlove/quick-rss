EchoMusic： 一个专为桌面端打造的简约、精致、功能强大的第三方音乐播放器

EchoMusic 是一款专为桌面端打造的第三方音乐播放器，兼顾简约设计与完整功能体验。它不仅提供流畅的音乐播放能力，还在歌词、评论、系统集成等细节上做了深度优化，带来更加沉浸和高效的听歌体验。同时采用直连官方服务的方式，避免第三方中转，进一步保障数据安全。支持 macOS、Windows 与 Linux，是一个真正面向桌面场景打造的全平台音乐播放器。

核心特性

  • 🎨 精致桌面设计 适配桌面布局，支持深浅色模式，兼顾美观与信息密度

  • 🔒 数据安全 官方服务直连，无第三方服务器中转

  • 🔎 音乐推荐与搜索 支持歌曲、歌手、专辑、歌单、排行榜推荐与全局搜索

  • ▶️ 完整播放控制 播放队列、播放模式切换、进度拖动、音量调节等

  • 💬 评论与互动 支持歌曲评论查看与楼层跳转

  • 📝 歌词系统 支持 LRC 解析、滚动同步、全屏歌词、桌面歌词

  • ⚙️ 系统集成 系统托盘、托盘控制、窗口控制、全局快捷键

  • 💾 本地持久化 设置、播放历史、收藏、播放状态本地保存

  • 🌍 跨平台支持 macOS / Windows / Linux 全平台适配

  • 🎧 音质与音效 支持 Hi-Res、FLAC、320kbps 等音质 内置多种音效（人声、DJ、环绕等）

Peninsula：开源的 macOS 刘海的窗口管理器和消息中心

Peninsula 是一款专为 macOS 设计的 刘海工具应用，帮助你更高效地管理 Mac。它集 窗口切换、通知管理、文件存储 于一体，让你的 Dynamic Peninsula（动态刘海）变得更加实用！

特色功能

✅ 增强版 Cmd-Tab 应用切换器

  1. 替代默认的 Cmd-Tab，提供更顺畅的多任务切换体验
  2. Cmd + Tab：切换到下一个应用
  3. Cmd + Shift + Tab：切换到上一个应用
  4. Cmd + Esc：退出切换器，不做任何操作

✅ 一键开启 Peninsula 窗口管理

点击刘海区域，即可快速调出 Peninsula 界面，再次点击切换不同功能窗口，点击刘海外部即可退出。

目前提供 5 种 Peninsula 窗口：

  1. 📌 应用切换器：显示当前屏幕所有运行的应用
  2. 🔔 通知中心：管理你关注的通知应用
  3. 📂 文件暂存区：快速存取临时文件
  4. 🔧 菜单：基础操作（如退出 Peninsula）
  5. ⚙️ 设置中心：个性化配置 Peninsula

✅ 智能通知管理

  1. 可自定义添加你最关心的通知应用
  2. 当有通知时，Peninsula 会在 刘海右侧显示图标，点击即可打开
  3. 6.18 秒后，自动变成 未读通知计数，点击数字即可进入通知中心
